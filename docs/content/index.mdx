# TazeAI Documentation

Welcome to the TazeAI documentation. TazeAI is an open-source "Cursor for Designers" that lets you visually edit React websites or webapps and write your changes back to code in real-time.

![TazeAI Hero Banner](/images/hero-banner.png)
*TazeAI empowers designers to make direct edits to React and TailwindCSS projects*
